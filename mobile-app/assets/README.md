# Recursos gráficos

Este directorio queda reservado para los iconos y pantallas de arranque que necesita Expo al generar builds de producción.

Incluye tus propios archivos PNG antes de crear un APK o enviar a las tiendas:

- `icon.png`: 1024x1024 px sin transparencia.
- `adaptive-icon.png`: 432x432 px con transparencia (Android).
- `splash.png`: se recomienda 1242x2436 px.

Ajusta las rutas en `app.json` si cambias los nombres o añades variantes para modo oscuro.
